# terraform-module-ec2

Reusable AWS EC2 module.

## Versioning

Module version is stored in the `VERSION` file.

Consumers should pin using Git tags.

```hcl
module "ec2" {
  source = "git::https://github.com/your-org/terraform-module-ec2.git?ref=v1.0.0"
}
```
